Custom Crit Sounds
==================
Drop .ogg sound files in this folder, then register them in-game:

  /cp addsound mysound.ogg

The sound will appear in the crit sound picker in /cp settings.
Use the Preview button to test it.

To remove a custom sound:
  /cp removesound mysound.ogg

Notes:
- Only .ogg format is supported by WoW
- The sound file length IS the play duration (WoW plays the full file)
- Trim your .ogg files to the desired length before adding them
- Free tool for trimming: Audacity (audacityteam.org)
